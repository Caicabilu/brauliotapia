# Portafolio Profesional - Braulio Nicolás Tapia Zuñiga

Este es el sitio web de mi portafolio profesional, diseñado con una estética moderna y elegante basada en el concepto de "Viento y Nubes". Como Administrador de sitios web y Desarrollador Web, este espacio muestra una selección de los proyectos más destacados que he gestionado y desarrollado.

## 🌤️ Concepto Visual
El sitio utiliza técnicas modernas de diseño para ofrecer una experiencia fluida y profesional:
- **Glassmorphism:** Uso de tarjetas translúcidas con efectos de desenfoque de fondo.
- **Nubes Animadas:** Un fondo dinámico con nubes que flotan suavemente para dar vida al sitio.
- **Minimalismo Elegante:** Enfoque en la tipografía (Outfit) y una paleta de colores celestial (azules y blancos).
- **Responsive Design:** Optimizado para dispositivos móviles y escritorio.

## 🛠️ Tecnologías Utilizadas
- **HTML5 Semántico**
- **CSS3 Vanilla** (Animaciones personalizadas, Flexbox/Grid, Backdrop Filters)
- **JavaScript Moderno** (Interacciones de scroll, Efectos parallax)
- **Vite** (Entorno de desarrollo rápido)

## 📁 Proyectos Destacados
El portafolio incluye capturas reales y enlaces a proyectos como:
- **ContainerFlow:** Gestión inteligente de contenedores marítimos.
- **Cabañas San Joaquín:** Sistema de arriendo en Isla Negra.
- **MAFRA TMI:** Taller mecánico corporativo.
- **South Star Seafood:** Portal de exportación premium.
- **IsapreparaTí**, **Angeli Músicos**, **Ingeman Flagos**, y **FonoTimbre**.

## 🚀 Instalación y Desarrollo
Si deseas ejecutar este proyecto localmente:

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/Caicabilu/brauliotapia.git
   ```
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Ejecutar servidor de desarrollo:
   ```bash
   npm run dev
   ```
